# 🎬 Movie & TV Search App (Java)

Aplicação desktop desenvolvida em Java que permite consultar informações de filmes e séries utilizando uma API externa.

## 🚀 Funcionalidades

- 🔍 Busca por nome de filmes ou séries
- 📅 Exibição de data de lançamento
- ⭐ Avaliação do conteúdo
- 🎬 Exibição de capa (poster)
- 📺 Informações adicionais para séries (temporadas e episódios)
- ⚡ Interface responsiva (sem travamentos)

## 🛠️ Tecnologias utilizadas

- Java
- Swing (Interface gráfica)
- HttpClient (requisições HTTP)
- JSON (manipulação de dados)
